ReadMe file for Arduino Wear Leveling Library v1.0
https://github.com/drjoju/Arduino-wear-leveling
Proengsoft Apr 2014


--------------------------------------------------------------------------------
To use the library:
(1) Go to https://github.com/drjoju/Arduino-wear-leveling and click the ZIP button to
download the repository as a ZIP file to a convenient location on your PC.
(2) Uncompress the downloaded file. This will result in a folder containing all
the files for the library, that has a name that includes the branch name,
for example "Arduino-wear-leveling-master".
(3) Rename the folder to just "Arduino-wear-leveling".
(4) Copy the renamed folder to the Arduino sketchbook\libraries folder.
